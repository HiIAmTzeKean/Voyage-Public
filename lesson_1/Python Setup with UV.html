What is UV?

  • Extremely fast Python package and project manager
  • Written in Rust - 10-100x faster than pip
  • All-in-one replacement for pip, venv, virtualenv, and pipx
  • Automatic Python version management

Why UV for Our Team?

  • Fast dependency resolution for data science packages
  • Consistent environments across team members
  • Simplifies Python version management
  • Modern lockfile-based workflow

Prerequisites

macOS

Terminal access (built-in)

Windows

PowerShell or Command Prompt (built-in)

No Python installation required - UV manages it for you!

Installing UV on macOS

Open Terminal and run the standalone installer:

curl -LsSf https://astral.sh/uv/install.sh | sh

Alternative: Install via Homebrew

brew install uv

Installing UV on Windows

Open PowerShell as Administrator and run:

powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"

Alternative: Install via WinGet

winget install --id=astral-sh.uv -e

Verify UV Installation

After installation, restart your terminal and verify:

uv --version

You should see output like:

uv 0.10.2

Installing Python with UV

UV automatically downloads Python when needed. To install specific version:

uv python install 3.12

Check installed Python versions:

uv python list

Creating a New Project

Initialize a new Python project:

mkdir my-project
cd my-project
uv init

This creates pyproject.toml and sets up project structure

Installing Jupyter Packages

Add Jupyter and essential data science packages:

uv add jupyter ipykernel notebook

Add common analysis libraries:

uv add pandas numpy matplotlib

Setting Up VS Code

  • Download VS Code from code.visualstudio.com
  • Install for your operating system
  • Launch VS Code after installation

Required VS Code Extensions

  • Python (by Microsoft) - Core Python support
  • Jupyter (by Microsoft) - Notebook support
  • Pylance (by Microsoft) - Language server

Install via Extensions panel (Ctrl+Shift+X / Cmd+Shift+X)

Open Project in VS Code

From terminal in your project directory:

code .

Or use VS Code menu:

File → Open Folder → Select your project folder

Creating a Jupyter Notebook

In VS Code, press Ctrl+Shift+P (Cmd+Shift+P on Mac)

Type and select:

Jupyter: Create New Blank Notebook

Save as notebook.ipynb

Selecting Python Kernel

  • Click "Select Kernel" in top-right of notebook
  • Choose "Python Environments"
  • Select the .venv environment from your project

The kernel name will show in top-right corner

Testing Your Setup

In a notebook cell, run:

import pandas as pd
import numpy as np
import matplotlib.pyplot as plt

print("Setup successful!")
print(f"Pandas version: {pd.__version__}")

Press Shift+Enter to execute

Essential UV Commands

  • uv add <package> - Install package
  • uv remove <package> - Remove package
  • uv sync - Sync dependencies
  • uv run python script.py - Run script

Common Issues

  • Kernel not found: Reinstall ipykernel
  • Import errors: Run uv sync
  • Wrong Python: Check kernel selection
  • VS Code not detecting .venv: Reload window

Best Practices

  • Always work within project directories
  • Use uv add instead of pip
  • Commit pyproject.toml and uv.lock to git
  • Keep UV updated with uv self update
