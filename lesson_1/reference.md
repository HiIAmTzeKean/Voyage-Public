
# Reference Resources

## Datasets

### Kaggle Dataset
- [Travel Insurance Dataset](https://www.kaggle.com/datasets/mhdzahier/travel-insurance) - A dataset for travel insurance analysis available on Kaggle.

## APIs

### Financial Data APIs
- [yfinance](https://github.com/ranaroussi/yfinance) - A Python library for downloading historical market data from Yahoo Finance.
- [Alpha Vantage](https://www.alphavantage.co/) - Free API for real-time and historical stock data, forex, and cryptocurrencies.
- [IEX Cloud](https://iexcloud.io/) - Provides financial market data with a free tier for developers.
- [Polygon.io](https://polygon.io/) - Real-time and historical market data API with free access for limited usage.
- [Nasdaq Data Link (formerly Quandl)](https://data.nasdaq.com/) - Platform for financial, economic, and alternative data with free datasets.
- [FRED API](https://fred.stlouisfed.org/docs/api/fred/) - Free API for economic data from the Federal Reserve Bank of St. Louis.

## Databases

- [DuckDB](https://duckdb.org/) - An in-process SQL OLAP database management system.
- [PostgreSQL](https://www.postgresql.org/) - A powerful, open-source object-relational database system.
- [Snowflake](https://www.snowflake.com/) - A cloud-based data warehousing platform for analytics and data storage.
- [MySQL](https://www.mysql.com/) - An open-source relational database management system.
- [Google BigQuery](https://cloud.google.com/bigquery) - A serverless, highly scalable, and cost-effective multi-cloud data warehouse.
- [Amazon Redshift](https://aws.amazon.com/redshift/) - A fully managed, petabyte-scale data warehouse service in the cloud.

## ETL Tools

- [Apache Airflow](https://airflow.apache.org/) - A platform to programmatically author, schedule, and monitor workflows.
- [Dagster](https://dagster.io/) - A data orchestrator for machine learning, analytics, and ETL.
- [Prefect](https://www.prefect.io/) - A workflow orchestration framework for data pipelines.
- [Luigi](https://github.com/spotify/luigi) - A Python module for building complex pipelines of batch jobs.
