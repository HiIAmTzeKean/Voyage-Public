# Assessment Rubric

**Project Evaluation (5-point scale per project):**

| Criterion              | Scoring                                      |
|------------------------|----------------------------------------------|
| Problem Framing       | Clarity of scope, defined metrics, business context |
| Technical Approach    | Appropriate methods, justified choices, ablations |
| Code Quality          | Modularity, testing, style, comments         |
| Experimentation       | Systematic approach, error analysis, insights |
| Deployment            | Working system, monitoring, documentation    |
| Portfolio Presentation| README clarity, reproducibility, demo        |

**Passing:** 3/5 (meets masters-level expectations) | **Strong:** 4-5/5 (recruiter-worthy)
